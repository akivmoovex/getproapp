# Spacing rules

- **Grid** — Core rhythm is **8px**: `--space-1` (8), `--space-2` (16), `--space-3` (24), `--space-4` (32), `--space-5` (40).
- **Touch** — Minimum interactive height `--space-touch` (48px) for primary taps.
- **Extended scale** — `--space-half`, `--space-1-5`, etc. are for fine-tuning; prefer the core steps for new layouts.

## Do not

- Use raw `px` / `rem` literals in component CSS when a token exists.
